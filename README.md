# Gambz Laundromat Management System

**Final Project for ICS2609: Application Development & Emerging Technologies 2 (Enterprise Back-End)**

A full-stack laundromat management system that provides comprehensive business management features including transaction processing, customer management, service tracking, analytics reporting, and staff administration.

## Tech Stack

- **Frontend:** Vue.js 3, Bootstrap Vue Next, Vite
- **Backend:** Node.js, Express.js
- **Database:** MySQL

## Key Features

- Transaction logging and management
- Customer management
- Service & addon management
- Analytics dashboard and reporting
- Edit history tracking
- CSV import/export functionality

Built with enterprise-grade architecture focusing on scalable back-end development, RESTful API design, and modern front-end frameworks.
